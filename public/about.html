What is "The Hexagon Of Life"?

The Hexagon Of Life is a computer programm that helps teachers teach and students study. The programm simulates evolution using herbivores and carnivores. The herbivores eat plants and the carnivores eat the herbivores. We created it in a very explaining way, so that it is a good way for students to understand evolution better

Everytime a new simulation starts, a new random map is created, based on the preferences of the user. There is an import and export button with the intent that teachers can share the settings they want with their pupils. This way the pupils can run the simulation the way the teacher thinks is best. When the simulation has started, a user can pause or stop it. The user can change settings, see statistics and watch the simulation play out. Once stopped, the user can et the statistics, and upon pressing play, a new simulation will start.
